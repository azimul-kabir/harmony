# Harmony

Harmony is a self-hosted music library manager designed for collectors who want complete control over their music.

It scans your local music library, extracts metadata, keeps a synchronized database of your collection, and helps identify which songs from Spotify playlists you already own and which are missing.

## Features

### Current

- FastAPI web application
- SQLite library database
- Automatic music library scanning
- Metadata extraction using Mutagen
- Library statistics
- Docker support

### Planned

- Spotify playlist analyzer
- Missing song detection
- SpotDL download queue
- Automatic library synchronization
- Duplicate detection
- Metadata repair
- Album artwork management
- Navidrome integration
- Jellyfin integration

## Technology

- Python 3.12
- FastAPI
- SQLAlchemy
- SQLite
- Mutagen
- Docker
